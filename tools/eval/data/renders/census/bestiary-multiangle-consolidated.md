# FLAME Multi-Angle Bestiary (Wave 16d Census)

Generated from 900 renders: 60 components × 5 values × 3 cameras (front, left34, closeup).
6 independent Gemini critics evaluated ψ0-29 and β0-29.

## Critical Corrections from Multi-Angle Analysis

These components were **misidentified** in the frontal-only census:

| Component | Old Label (frontal) | Corrected Label (multi-angle) | Why Frontal Was Wrong |
|-----------|--------------------|-----------------------------|----------------------|
| **ψ4** | boredom ↔ engagement | Brow Raise ↔ Brow Furrow | Emotional read was correct *direction* but mechanical is more precise |
| **ψ8** | flat/bored ↔ shocked | **Lateral Jaw Shift (L↔R)** | Frontal view hid the lateral displacement — looked like an emotional change |
| **ψ9** | frown ↔ smile | **Eye Squint/Close ↔ Eye Wide Open** | Frontal view conflated eye changes with overall expression |
| **ψ3** | disgust ↔ open curiosity | Mouth Stretch Wide ↔ Lip Pucker | Frontal flattened lip protrusion at +3 |
| **ψ6** | surprise ↔ angry | Rounded Mouth "Oh" ↔ Horizontal Lip Stretch | left34 shows trumpet-like protrusion at -3 |

### Impact on Current Recipes

- **ψ8 at 3.5 in ALARM_ALARMED**: We're using a lateral jaw shift as our "shocked" signal. The emotional read from front *works* (the face looks startled), but the mechanism is jaw displacement, not the "shocked eyes" we assumed.
- **ψ9 at -1.5 in AGGRESSION**: Correctly used — we wanted squint, and that's what ψ9 negative does.
- **ψ9 at -2.0 in FATIGUE_EXHAUSTED**: Correctly used — we wanted "frown"/depletion, but the actual mechanism is eye squint which still reads as "tired" in context.

## Methodology Note

The original bestiary (frontal-only) was developed from interactive explorer testing with a single observer. This multi-angle census used 900 static renders evaluated by 6 independent Gemini critics, each covering 10 components across 3 camera angles. The key value of multi-angle analysis is catching depth/protrusion changes invisible from front (especially ψ26 chin resolve, β5 sagittal profile, β15 chin projection).
